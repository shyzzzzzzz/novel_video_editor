# VibeStudio PRD

**产品名称**: VibeStudio
**日期**: 2026-04-11
**状态**: 草稿

---

## 一句话定位

电影工业级 AI 短剧生成桌面工具 — 输入 vibe，AI 完成创作全流程，用户只专注讲故事。

---

## 用户价值

- 极简输入，AI 接管电影级制作复杂度
- 任何层级可深入，但从不强迫用户进入层级
- 专业级输出，门槛接近于零

---

## 用户视角架构

| 入口 | 内容 |
|------|------|
| **创作 (Create)** | 当前任务 — Vibe 输入 / 剧本 / Takes / 剪辑 |
| **项目 (Project)** | 所有集/季总览 + 快速跳转 |
| **资产 (Assets)** | 角色库 / 场景库 / 模板库 — 按需浮现 |

**设计原则**: 用户不主动进入层级，层级自动导航用户。复杂的东西藏在背后，少即是多。

---

## 系统层级架构（对用户隐藏）

```
工作区 → 项目 → 剧集 → 分季 → 分集 → 剧本 → 分镜 → 场景
```

每层都可编辑，但用户在任何时候都不需要主动"选择自己在哪里"——上下文自动导航。

---

## 核心功能

### 1. Vibe 输入层（默认视图）

- **Prompt 输入**: 多行自然语言，支持氛围/情绪/风格描述
- **参考素材**: 图片（角色/场景参考）、音频（音乐/氛围）、视频（风格参考）
- **主题模板**: 预设模板库（剧情短片/广告/宣传片/教学案例等）
- **生成模式**:
  - 全自动模式：AI 端到端完成，用户等待
  - 协作模式：每步（剧本/角色/场景）暂停，用户确认或修改后继续

### 2. 剧本管理

- 多层级剧本编辑（对应项目层级结构）
- 版本化控制，任意节点可回溯
- AI 辅助生成与修改

### 3. 角色系统

- **角色卡**: 图片 + 关键描述（供 AI 生成时引用保持一致性）
- **模板 + 变体机制**: 主角全局一致，配角可换
- **跨集一致性管理**: 版本快照 + 强制引用模式

### 4. 场景管理

- 混合模式：资产库预建 + AI 实时生成
- 常用场景预建，不常用/一次性场景 AI 实时创建
- 场景库跨项目可复用

### 5. 分镜管理

- 剧本 → AI 生成分镜建议（镜头拆分 + 画面描述）
- 用户确认/调整每个镜头
- 可视化分镜面板

### 6. Takes 系统

- 同一镜头 AI 生成多个版本（不同风格/构图/表演）
- 版本对比选择界面
- 最佳版本进入剪辑线

### 7. 渲染/生成

- API 调用（图像 + 视频生成）
- 任务队列管理
- 进度追踪与状态显示

### 8. 审片/Approval 工作流

- Daily 素材审阅
- 批注和意见追踪
- 版本标记（Draft v1 / v2 / 终版）

### 9. 音频管线

- 对话/配音（BGM）
- 背景音乐
- 音效（Foley）
- 混音
- 音视频同步管理

### 10. 剪辑合成

- AI 指导下的专业剪辑
- 多轨道、时间线编辑、特效
- 本地完成

### 11. 资产复用

- 跨项目资产库
- 从旧项目继承角色/场景到新项目

### 12. 导出

- 本地视频导出

---

## UI 哲学

- **全屏沉浸**: 任何时候核心内容最大化
- **浮动面板**: 模块以浮动/可收起方式存在，按需浮现
- **极简设计**: 只让用户关注自己需要的信息，只让 AI 关注 AI 需要的信息
- **上下文导航**: 用户不需要选择"自己在哪个层级"，系统自动感知当前位置
