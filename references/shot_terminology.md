# 镜头术语参考

## 景别（Framing）

| 中文 | 英文 | 说明 |
|------|------|------|
| 大特写 | Extreme Close-up (ECU) | 极度接近目标，如单只眼睛、嘴唇 |
| 特写 | Close-up (CU) | 头部/脸部完整呈现 |
| 中近景 | Medium Close-up (MCU) | 头部到胸口 |
| 中景 | Medium Shot (MS) | 膝盖以上 |
| 中远景 | Medium Wide Shot (MWS) | 大腿到脚 |
| 远景 | Wide Shot (WS) | 全身，人与环境关系 |
| 大远景 | Extreme Wide (EWS) / Establishing Shot | 广阔环境，人如蚂蚁 |

## 运镜方式（Camera Movement）

| 中文 | 英文 | 说明 |
|------|------|------|
| 固定 | Static / Locked Off | 镜头不动 |
| 推 | Dolly In / Push In | 靠近主体 |
| 拉 | Dolly Out / Pull Out | 远离主体 |
| 摇 | Pan | 水平转动 |
| 倾 | Tilt | 垂直转动 |
| 移 | Tracking / Dolly | 跟随主体移动 |
| 跟 | Follow | 持续跟随主体 |
| 升降 | Crane Up/Down | 垂直升降 |
| 旋转 | Roll | 镜头自转 |
| 晃动 | Handheld | 手持不稳定感 |

## 构图方式（Composition）

- **三分法 (Rule of Thirds)**: 将画面横竖各三等分，主体放在交叉点上
- **视线引导 (Lead Room)**: 主体视线前方留白
- **框架 (Framing)**: 用门窗等自然框架构图
- **对称 (Symmetry)**: 上下或左右对称
- **对角线 (Diagonal)**: 利用对角线引导视线

## 光影设计（Lighting）

| 类型 | 说明 |
|------|------|
| 顺光 (Front Light) | 光线来自主体正面，柔和均匀 |
| 侧光 (Side Light) | 光线来自侧面，制造立体感 |
| 逆光 (Back Light) | 光线来自主体背后，轮廓光效果 |
| 顶光 (Top Light) | 光线来自正上方 |
| 底光 (Bottom Light) | 光线来自下方，阴森效果 |
| 伦勃朗光 (Rembrandt) | 45度侧上光，脸部三角光斑 |
| 蝴蝶光 (Butterfly) | 正上方偏前的美人光 |

## 转场方式（Transitions）

| 中文 | 英文 | 说明 |
|------|------|------|
| 切 | Cut | 硬切 |
| 淡入淡出 | Fade In/Out | 渐变 |
| 叠化 | Dissolve / Cross Fade | 慢切换 |
| 划入划出 | Wipe | 滑动切换 |
| 溶黑 | Dip to Black | 中间黑过渡 |
